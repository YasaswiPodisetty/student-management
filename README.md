# Student Management System

## Tech Stack
- Java Spring Boot
- MySQL
- JPA / Hibernate

## Features
- Add student
- View students
- Update student
- Delete student

## API Endpoints
- GET /students
- POST /students
- PUT /students/{id}
- DELETE /students/{id}
